{% include 'header.html' %} This is a proof of concept metadata_statements signing service mainly targeted to Federation Operator, but it can also be used by any intermediate entities between the federation and the Relying Parties or OpenID Providers.

Federation

  • Check federation public key
  • Add a hierarchical superior entity
  • List superior entities

Entities

  • Enrol a new entity into the federation
  • List/delete enrolled entities

Tools

  • Generate JWKS

APIs

  • /getms/<issuer_urlsafe>

    Returns all the metadata statements for the requested issuer.

  • /getms/<issuer_urlsafe>/<fo_urlsafe>

    Returns the metadata statements for the requested issuer, but this time only those signed by the requested federation operator.

  • /getsk/<issuer_urlsafe>

    Returns the signing keys for the requested issuer.
